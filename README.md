# Академический Салон — платформа

Сайт-лидогенератор мастерской академических работ (дипломы, диссертации, статьи).
Статичный сайт без сборщика: HTML + CSS + ванильный JS; 3D-фон и анимации подключаются с CDN.
Эстетика «Полночный салон» (midnight + gold).

## Страницы
- `index.html` — главная (hero, ценности, процесс, тарифы, FAQ)
- `configurator.html` — конфигуратор заказа: живой расчёт цены, профиль эксперта, план, бронь слота
- `dashboard.html` — личный кабинет с трекингом заказа по этапам
- `check.html` — бесплатный инструмент «Проверка текста» (эвристический анализ риска ИИ)
- `referral.html` — реферальный клуб, бонусы и статусы
- `expertise.html` — витрина экспертизы по дисциплинам
- `knowledge.html` — база знаний / блог
- `404.html` — страница ошибки

## Структура
- `assets/css/styles.css` — дизайн-система (токены и компоненты)
- `assets/js/app.js` — общий слой: шапка/подвал, 3D-фон, UX-утилиты (тосты, store, валидация), логика калькулятора
- `assets/img/` — favicon, OG-обложка, apple-touch-icon
- `robots.txt`, `sitemap.xml` — SEO

## Локальный запуск
Любой статичный сервер, например:

```bash
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

Либо просто открыть `index.html` в браузере.

## Деплой
Чистая статика — встаёт на любой хостинг (nginx, Timeweb, Netlify, Vercel, Cloudflare Pages).

## Особенности реализации
- Прогрессивное улучшение: контент виден без JS; уважается `prefers-reduced-motion`.
- Three.js грузится лениво (вне критического пути), с фолбэком на градиент при отсутствии WebGL.
- Плавные переходы между страницами (View Transitions API) + предзагрузка ссылок.
- Демо-данные (заказы, бонусы) хранятся локально в браузере (`localStorage`).

Бэкенд (онлайн-оплата, база данных, реальные аккаунты и рефералка) — следующая фаза.
