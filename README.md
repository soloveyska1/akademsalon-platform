# Академический Салон

Сайт-лидогенератор мастерской академических работ (дипломы, диссертации, статьи).
Статичный сайт без сборщика: HTML + CSS + ванильный JS. Без внешних скриптов — только Google Fonts.

Дизайн-направление — **«Оттиск»** (университетская пресса): тёплая бумага, антиква Literata,
гротеск Golos Text, моноширинные цифры JetBrains Mono, тонкие линейки, строки с отточиями,
сургучный акцент, SVG-печати. Принцип движения: анимируется только то, что «печатается,
пишется или считается».

## Страницы
- `index.html` — главная: манифест + мгновенная смета, § 01 процесс и оплата, § 02 каталог цен, § 03 гарантии с примером отчёта, § 04 бесплатная проверка, § 05 FAQ
- `configurator.html` — «бланк заказа»: 4 шага, живая смета-ledger, черновик в localStorage, экран успеха со штемпелем
- `tariffs.html` — прайс-каталог: лесенка 11 типов, открытая формула цены, уровни сопровождения, «Разбор плана» (трипваер), FAQ о деньгах
- `guarantees.html` — «Устав мастерской»: 5 параграфов гарантий, шкала этапной оплаты, блок «чего мы не обещаем»
- `check.html` — бесплатная проверка текста на канцелярит и машинальность (анализ в браузере, текст никуда не уходит)
- `dashboard.html` — личный кабинет-«формуляр»: этапы, оплата, отчёт о проверках (демо на localStorage)
- `404.html` — страница ошибки
- `referral.html`, `expertise.html`, `knowledge.html` — redirect-заглушки на главную (разделы упразднены при редизайне)

## Структура
- `assets/css/styles.css` — дизайн-система «Оттиск» (токены и компоненты: sheet, dotrow, ledger, paysteps, report-sheet, seal, tag…)
- `assets/js/app.js` — общий слой: ценовая логика (`window.SalonCalc` — единственный источник цен), UX-утилиты (`Salon.*`), инжекция колонтитула/оглавления/подвала, reveal, печати `[data-seal]`
- `assets/img/` — favicon, OG-обложка, apple-touch-icon
- `robots.txt`, `sitemap.xml` — SEO

## Контракты localStorage
- `salon_draft` — черновик заказа: `{ state: {type, disc, term, tier}, idx, savedAt }`
- `salon_orders` — заявки: `{ id, createdAt, typeId…tierLabel, priceLow, priceHigh, topic, contact, stageIndex }`

## Локальный запуск
```bash
node .claude/static-server.js   # http://localhost:8000
# либо python3 -m http.server 8000
```

## Деплой
Чистая статика — встаёт на любой хостинг (nginx, Timeweb, Netlify, Vercel, Cloudflare Pages).

## Особенности реализации
- Контент виден без JS; шапка/подвал инжектируются прогрессивно; `prefers-reduced-motion` уважается везде.
- View Transitions API для переходов между страницами (с фолбэком), speculation rules / prefetch для мгновенной навигации.
- Шрифты — Google Fonts с кириллицей: Literata (variable), Golos Text, JetBrains Mono.
- Бэкенда нет: заявки пишутся в localStorage, демо-состояния честно помечены. Онлайн-оплата, реальные аккаунты и уведомления — следующая фаза.
