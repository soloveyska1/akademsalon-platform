# План проверки конверсии

## Правила

- Сначала исправляются ошибки и несоответствия, затем запускаются эксперименты.
- Одновременно меняется один смысловой фактор.
- Нельзя оптимизировать только клики: победитель определяется по
  квалифицированным заявкам и оплате допустимого продукта.
- В каждом тесте контролируются возвраты, жалобы, неподходящие запросы и
  ошибочные ожидания клиентов.
- Фальшивый дефицит, скрытые условия, заранее отмеченные допуслуги и
  подтверждающие манипуляции не тестируются.

## События

Минимальный словарь без содержимого полей и персональных данных:

- `hero_primary_click`;
- `hero_example_click`;
- `route_stage_selected`;
- `route_need_selected`;
- `estimator_started`;
- `estimator_completed`;
- `application_started`;
- `application_submitted`;
- `application_qualified`;
- `application_redirected`;
- `application_rejected`;
- `diagnostic_offer_sent`;
- `diagnostic_paid`;
- `support_offer_sent`;
- `support_paid`;
- `refund_requested`;
- `review_consent_given`;
- `review_published`.

## Очерёдность тестов

### E01. Первый экран

Вариант A:

> Курсовая или диплом зашли в тупик?

Вариант B:

> Помощь с курсовой, ВКР и дипломом — по шагам

Основная метрика: квалифицированная заявка на 100 уникальных посетителей.
Защитная метрика: доля заявок с ожиданием «сделать за меня».

### E02. Основной призыв

Вариант A: «Разобрать мою задачу».
Вариант B: «Получить план действий».

Метрика: завершение короткой анкеты, затем квалифицированная заявка.

### E03. Вход по этапу

Вариант A: сначала выбрать вид работы.
Вариант B: сначала выбрать состояние — «тема», «черновик», «замечания»,
«скоро защита».

Метрика: завершение анкеты и время до релевантной рекомендации.

### E04. Диагностика

Вариант A: самостоятельная недорогая услуга с полным зачётом.
Вариант B: бесплатный первичный просмотр и платный подробный разбор.

Метрики: оплата диагностики, переход в сопровождение, возвраты, нагрузка мастера.

### E05. Пакеты

Вариант A: «Диагностика / Редакторский аудит / Сопровождение».
Вариант B: «План действий / Сильный черновик / До защиты».

Метрики: правильный выбор пакета и число ручных переквалификаций.

### E06. Доказательство качества

Вариант A: обезличенный пример редакторского отчёта.
Вариант B: короткий кейс «было → замечание → решение клиента».

Метрика: квалифицированная заявка после просмотра доказательства.

## Условие остановки

Эксперимент останавливается немедленно, если он:

- увеличивает долю запрещённых запросов;
- создаёт ложное ожидание готовой аттестационной работы;
- ломает сохранение согласия, цены или версии документов;
- повышает возвраты или обоснованные жалобы;
- требует передавать содержимое формы в аналитику.
